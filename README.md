# Portfolio site for saeedzns.github.io

## How to use

1. Create a new GitHub repository named exactly:

```bash
saeedzns.github.io
```

2. Copy these files into that repository:

```text
index.html
styles.css
```

3. Push to GitHub.

4. In GitHub, go to:

```text
Settings -> Pages -> Build and deployment -> Deploy from branch -> main / root
```

5. Your site should become available at:

```text
https://saeedzns.github.io
```

## What to customize

Search for `TODO` in `index.html` and replace demo links after you deploy Streamlit/Hugging Face apps.
