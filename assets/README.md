# Assets Folder

Add `Saeed_Zohoorian_CV.pdf` here to enable the "Download CV" button on the portfolio.

To generate or add your CV:
1. Export your CV as PDF from Google Docs, Word, or your preferred editor
2. Name it `Saeed_Zohoorian_CV.pdf`
3. Place it in this `assets/` folder
4. Commit and push to GitHub

The portfolio will automatically link to it.
